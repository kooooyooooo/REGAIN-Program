## Step1：要求定義 💭

**「こんなアプリがほしい！」を言葉にするステップです**

まずは頭の中にあるアイデアを整理して、Geminiに伝えましょう！✨

あなたの「こうしたい」という想いを自由に書いてみてください。完璧じゃなくても大丈夫。AIが一緒に形にしてくれますよ 🙌

geminiに以下のプロンプトを入力

```markdown
私は以下のようなコーチです。以下のようなアプリを作りたいので要求定義書を作成してください。

目的：(どういった目的かを記載)

アプリの概要：(どういったアプリを作成したいか記載)

アウトプットはマークダウン形式をplain textのコードブロックで返して。
デモアプリの作成なので、最小限の構成にしてください。

------
(基本情報ヒアリングシートの内容を添付)
```

## Step2：要件定義 📋

**「ほしい」を「作れる形」に変換するステップです**

Step1で整理した想いを、実際に開発できる具体的な仕様書に落とし込みます 🎯

Geminiがあなたのアイデアを、エンジニアに伝わる言葉に翻訳してくれるイメージです！

geminiに以下のプロンプトを入力

```markdown
素晴らしいです。
要求定義書をもとに、要件定義書を作成してください。

デモアプリとして作成するので、バックエンドの実装は最低限にとどめてください。
```

## Step3：開発 🚀

**いよいよアプリが動き出す瞬間です！**

ここまで準備したら、あとはGensparkにお任せ 💪

要求定義と要件定義を渡すだけで、AIが実際に動くアプリを作ってくれます。ワクワクしますね！

- gensparkを開く
- 「AIデベロッパー」を選択
- 「シンプルなウェブアプリ」を選択
- モデルは「Claude Opus4.5」を選択
- 以下のプロンプトを入力
    
    ```markdown
    以下のアプリを作成してください
    (要求定義書を貼り付け)
    (要件定義書を貼り付け)
    ```
